# TurboNasirator3000
TurboNasirator3000 is an application for hard-wiring folders on the desktop

## Это приложение для жесткого насирания папокам на рабочий стол.
# 4 основных функции:
- Создание n-ого количества папок
- Удаление созданных папок вместе с их содержимым
- Упаковка данных, путём прятанья их среди созданных папок
- Распаковка данных, спрятанных среди папок
